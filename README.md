# Crowdfunding

# Démarrer le projet
Entrer la commande "java -jar" avec le nom du projet et son chemin.
Exemple : java -jar classes\artifacts\CrowdSourcingTool_jar\CrowdSourcingTool.jar

# Peupler le projet
Importer un fichier JSON avec la commande "import -f" et le nom du fichier JSON plus le chemin (il est préférable d'utiliser un fichier situé dans le dossier où vous avez ouvert l'invité de commande).
Exemple : import -f donations.json
